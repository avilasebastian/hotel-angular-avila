# HotelAngularAvila

Este proyecto usa:

    Angular CLI: 8.3.19
    Node: 10.16.3
    OS: win32 x64
    Angular: 8.2.14

Luego de descargar o clonar el proyecto debes pararte en tu proyecto en la terminal y ejecutar:

```npm install```

para descargar todas las dependencias del proyecto. 
Y luego:

```ng serve --open```
